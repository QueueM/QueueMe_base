Getting Started

Follow these steps to quickly integrate QueueMe API into your application

1

Register for API Access

Sign up for a QueueMe developer account to receive your API credentials. You'll get a set of development keys to start building and testing your integration.

Register Now
2

Authenticate with the API

Use your API credentials to obtain an access token. This token will be used to authenticate all your API requests. 

curl -X POST "https://api.queueme.net/api/v1/auth/token" \
     -H "Content-Type: application/json" \
     -d '{
       "client_id": "YOUR_CLIENT_ID",
       "client_secret": "YOUR_CLIENT_SECRET"
     }'
3

Make Your First API Call

With your access token, you can now make API calls. Let's start by retrieving a list of shops. 

curl -X GET "https://api.queueme.net/api/v1/shops" \
     -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
     -H "Content-Type: application/json"
4

Implement Key Features

Start implementing key features like bookings, queues, and notifications. Here's an example of creating a booking: 

curl -X POST "https://api.queueme.net/api/v1/bookings" \
     -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{
       "service_id": 123,
       "specialist_id": 456,
       "date": "2025-06-15",
       "time_slot": "14:30",
       "customer_id": 789
     }'
5

Go Live

Once you've thoroughly tested your integration, request production API keys and deploy your application. Our team will review your integration and provide feedback if needed.

Request Production Access

Official Python client library for QueueMe API. Perfect for backend systems, data analysis, and AI integrations.

JavaScript SDK

JavaScript client library for browser and Node.js applications. Easily integrate QueueMe into your web apps.
