Getting Started Guide

Authentication

QueueMe API uses JWT tokens for authentication. First, obtain a token by making a POST request to the /api/auth/login endpoint with your credentials.

Using Your Token

Include the token in the Authorization header of all subsequent requests: Authorization: Bearer your_token_here

Making API Calls

Now you can make requests to any API endpoint for which you have permission. See the Swagger UI or ReDoc for the complete list of available endpoints.

  const response = await fetch('https://api.queueme.net/api/auth/login/', {

    })
  });
  const data = await response.json();
  return data.access;
}
